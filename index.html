Use the Arrow keys to rotate the cube along the X- and Y-axes.

Use 4 and 6 to rotate the cube along the Z-axis.

Use S to scramble the cube.

Use U to undo the last rotation.

Use T (resp. Shift+T) to rotate the top layer clockwise (resp. anti-clockwise).

Use B (resp. Shift+B) to rotate the bottom layer clockwise (resp. anti-clockwise).

Use L (resp. Shift+L) to rotate the left layer clockwise (resp. anti-clockwise).

Use R (resp. Shift+R) to rotate the right layer clockwise (resp. anti-clockwise).

Use F (resp. Shift+F) to rotate the front layer clockwise (resp. anti-clockwise).

Use B (resp. Shift+B) to rotate the back layer clockwise (resp. anti-clockwise).
